1. ما هي Claude-Skills؟
───────────────────────────────────────────────────────────────────────────────

Claude-Skills هي 66 مهارة متخصصة لمطوري Full-Stack تعمل داخل Claude Code.
تُحوّل Claude من مساعد عام إلى "مطور خبير" في مجال محدد.

التثبيت:
    /plugin marketplace add jeffallan/claude-skills
    /plugin install fullstack-dev-skills@jeffallan

3. سير العمل المتعدد المهارات (Multi-Skill Workflows)

Claude-Skills تدعم سير عمل تلقائي يُنشَّط بناءً على السياق:

┌─────────────────────────────────────────────────────────────────────────────┐
│ سير عمل تطوير الميزة (Feature Development)                                  │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  Feature Forge → Architecture Designer → Fullstack Guardian → Test Master │
│       ↓              ↓                ↓              ↓                        │
│  اكتشاف المتطلبات  تصميم البنية    تنفيذ الكود    اختبار شامل              │
│                                                                             │
│  مثال: "أضف ميزة التنبؤ بالمبيعات"                                          │
│  → Feature Forge: تحليل المتطلبات وتقسيمها                                │
│  → Architecture Designer: تصميم نموذج التنبؤ + تدفق البيانات               │
│  → Fullstack Guardian: تنفيذ الكود (Apps Script + Sheets)                   │
│  → Test Master: اختبار النموذج بدقة 85%+                                   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

│                                                                             │
│  Debugging Wizard → Framework Expert → Test Master → Code Reviewer        │
│       ↓              ↓              ↓              ↓                        │

4. Context Engineering مع /common-ground
───────────────────────────────────────────────────────────────────────────────

/common-ground هو أمر خاص يُظهر افتراضات Claude المخفية عن المشروع:

الاستخدام:
    /common-ground

النتيجة المتوقعة:
    ┌─────────────────────────────────────────────────────────────┐
    │ Claude يفترض أن:                                              │
    │ • النظام يعمل على Google Workspace فقط (لا Azure/AWS)       │
    │ • AppSheet كافي للواجهة (لا حاجة لـ React)                  │
    │ • Apps Script يكفي للأتمتة (لا حاجة لـ Python)              │
    │ • البيانات صغيرة (<10,000 سجل)                              │
    │ • الفريق 5-10 موظفين                                        │
    │ • لا حاجة للتكامل مع ERP خارجي                               │
    └─────────────────────────────────────────────────────────────┘

الفائدة: كشف الافتراضات الخاطئة قبل البدء في التنفيذ!

8. الخلاصة
───────────────────────────────────────────────────────────────────────────────

Claude-Skills تحوّل Claude Code من "مساعد عام" إلى "فريق هندسي كامل":

  • Secure Code Guardian = مهندس أمان
  • API Designer = مهندس Backend
  • Frontend Expert = مهندس Frontend  
  • Data Analyst = عالم بيانات
  • DevOps Engineer = مهندس DevOps
  • Test Master = مهندس جودة
  • Code Reviewer = مراجع كود

النتيجة: نظام مبيعات احترافي، آمن، قابل للتوسع، ومُختبر.
