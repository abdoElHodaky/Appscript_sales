# قواعد السلوك / Code of Conduct

## التعهد / Pledge

نحن نتعهد بجعل المشاركة في مشروعنا تجربة خالية من المضايقة للجميع.

We pledge to make participation in our project a harassment-free experience for everyone.

## المعايير / Standards

### سلوك إيجابي / Positive Behavior

- استخدام لغة ترحيبية وشاملة
- احترام وجهات النظر المختلفة
- قبول النقد البناء بهدوء
- التركيز على ما هو الأفضل للمجتمع

### سلوك غير مقبول / Unacceptable Behavior

- استخدام لغة جنسية أو عنيفة
- التحرش بأي شكل
- نشر معلومات شخصية دون إذن
- سلوك غير مهني آخر
